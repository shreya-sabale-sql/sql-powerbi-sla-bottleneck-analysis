# SQL Analysis Files

This folder contains SQL scripts used to analyze IT ticket lifecycle data.

## File Overview
- `01_process_visibility.sql` – High-level process flow and resolution time analysis  
- `02_bottleneck_analysis.sql` – Identification of systemic delays and outliers  
- `03_sla_breach_analysis.sql` – SLA compliance analysis by stage, team, and priority  
- `04_team_capacity_analysis.sql` – Workload and capacity assessment for teams  

Each script is written with business context and is intended to be reusable
for similar operational analytics problems across industries.
